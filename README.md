## Run locally
###### start server
1. `. venv/bin/activate`
2. `cd server && flask run`
3. Navigate to http://localhost:5000/data and confirm data is being served

###### start app
1. `cd app`
2. `yarn install`
3. `yarn start`
4. Navigate to http://localhost:3000/
